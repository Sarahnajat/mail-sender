# Email Sender Script


A simple and interactive script to send emails directly from your terminal using **Node.js**!

# Built With
* Node.js
* Inquirer.js - A package for command-line prompts
* Nodemailer - A package for sending emails



## ⚠️ Important Note for Gmail Users
If you have enabled **2-Step Verification** (Two-Factor Authentication) on your Gmail account, you cannot use your regular email password to send emails through this script. Instead, you need to generate an **App Password** For more information, visit the official [Google Support Page](https://support.google.com/accounts/answer/185833?hl=en). .
